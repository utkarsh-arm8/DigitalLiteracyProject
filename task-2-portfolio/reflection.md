# Task 2 — Student Digital Portfolio

## Platforms Set Up

### 1. GitHub (github.com)
GitHub is the world's largest platform for hosting code, version control, and open-source collaboration. Developers use it to store projects, contribute to others' repositories, and showcase their programming work. I created my profile README repository (username/username) with my name, branch, year, and a brief learning goal statement.

### 2. LinkedIn (linkedin.com)
LinkedIn is a professional networking platform used by employers, recruiters, and academics worldwide. It functions like a living, digital CV. I filled in my Education section with: B.Tech (CSE), VIT Bhopal University, Expected Graduation 2028.

### 3. Kaggle (kaggle.com)
Kaggle is a platform for data science and machine learning enthusiasts. It hosts datasets, competitions, and notebooks. It is widely used by students and professionals to practice data analysis and build a public portfolio of ML projects.

## How I Plan to Use These Platforms Over Four Years

- **Year 1:** Build foundations — set up profiles, complete beginner challenges on Kaggle, and push small lab assignments to GitHub.
- **Year 2:** Start contributing to open-source projects on GitHub and participate in entry-level Kaggle competitions.
- **Year 3:** Use LinkedIn to connect with alumni, attend virtual events, and apply for internships. Update GitHub with major project work.
- **Year 4:** Leverage all three platforms for campus placements — a filled GitHub, a few Kaggle medals, and a strong LinkedIn profile create a compelling digital presence for recruiters.
